## 介绍
SpiderMAN 是基于Django+scrapyd+bootstrap的分布式爬虫管理系统，能实现分布式任务调度，对数据的监控等

## 功能
* 用户登录
* 管理服务器节点
* 管理爬虫、爬虫部署
* 查看数据统计
* 容器快速部署
